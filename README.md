# 元素尖塔 2 Demo

Vite + React 多文件版 Demo。

## 运行

```bash
npm install
npm run dev
```

浏览器打开终端显示的本地地址，通常是：

```text
http://localhost:5173
```

## 当前功能

- 回合制卡牌战斗
- 抽牌 / 弃牌 / 洗牌
- 元素附着
- 蒸发 / 融化 / 超载 / 冻结 / 扩散
- 战斗奖励三选一
- 跳过奖励
- 金币
- 路线选择
- 休息回血
- Boss 通关
- 自检面板
